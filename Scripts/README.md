# Simón Says - Trabajo Final IPW 2025

Bienvenido al proyecto **Simón Says**. Este es el Trabajo Final Integrador para la materia Introducción a la Programación Web (UAI).

##  Cómo Jugar

1. **Ingresa tu nombre**: Introduce un nombre de al menos 3 caracteres para comenzar.
2. **Mira la secuencia**: El juego iluminará un color al azar. Debes recordar el orden.
3. **Repite la secuencia**: Haz clic en los botones en el mismo orden.
4. **Sube de nivel**: Cada vez que aciertes la secuencia completa, se agregará un nuevo color y subirás de nivel.
5. **Game Over**: Si te equivocas, el juego terminará, se guardará tu puntaje en el ranking y podrás reiniciar la partida.

##  Estructura del proyecto

* **index.html**: Estructura principal del juego y el tablero.
* **contacto.html**: Formulario de contacto con validaciones.
* **Styles/**: Carpeta con los estilos CSS (incluyendo `reset.css`, `style.css` y `style-modal.css`).
* **Scripts/**: Carpeta con la lógica en JavaScript (ES5).

##  Características Especiales (Requerimientos)

* **Ranking local**: Los puntajes se guardan en el navegador usando `LocalStorage`.
* **Validaciones estrictras**: Formulario de contacto validado con expresiones regulares.
* **Diseño Responsivo**: Adaptado para diferentes tamaños de pantalla usando Flexbox.
* **Código Limpio**: Siguiendo las normas de la cátedra (sin líneas en blanco, funciones nominadas y orden de declaración).
